# Nodokļu kalkulators (Latvija)

Šī ir vienkārša Python programma darba algas nodokļa aprēķinam Latvijā. Tā ņem vērā standarta nodokļu likmes un sociālās iemaksas.

## Kā izmantot

1. Palaidiet programmu, izpildot `python calculator.py` komandrindā.
2. Ievadiet savu algu, kad jums tiks pieprasīts ievadīt datus.
3. Programma automātiski aprēķinās un parādīs ienākuma nodokļa, sociālo iemaksu un tīrās algas summas.

## Nodokļu likmes un atvieglojumi

- Ienākuma nodoklis tiek aprēķināts, pamatojoties uz fiksētu likmi 21%.
- Standarta nodokļa atvieglojums ir iestatīts 500.0 eiro apmērā.

## Piezīme

Pārliecinieties, ka izmantojat aktuālos nodokļu likmju un noteikumu datus Latvijai, jo tie var mainīties laika gaitā.
